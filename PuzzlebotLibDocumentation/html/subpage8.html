How reflectance sensors work

The puzzlebot firmware provides functionality to make use of reflectance type light sensors, such as the Pololu QTR-MD-16RC. These sensors make use of an inbuilt capacitor which discharges through a phototransistor. The more light incident on the transistor, the lower its resistance, and the faster the capacitor discharges. This results in a specific set of operations to read from a sensor:

  • Set pin as output;
  • Set the pin to output high to charge the capacitor;
  • After a short period, set the pin to an input;
  • The capacitor starts discharging;
  • Measure the time the voltage takes to fall to zero, which is the sensors output;

The sensors have a set of emitters. These emit light which is reflected by bright surfaces (such as a white line, and are controlled by 2 pins, such that alternate emitters are connected together.

Implementation

The following parameters have to be set when using the ReflectanceSensor class:

  • SensorPins: up to 16 sensors are suported;
  • SensorCount: number of light sensors present on the robot;
  • Timeout: reflect signal timeout (microseconds);

Example

#include "Arduino.h"
#include "ReflectanceSensor.h"
//Define the global variables to configure the reflectance sensor
uint8_t SensorCount = 6; // Number of refectance sensors
uint8_t SensorPins[6] = {23, 22, 19, 27, 25, 32}; // Sensor pins
uint32_t Timeout = 2500; // Sensor reflect timeout (us)
ReflectanceSensor sensor;
void setup()
{
sensor.SetSensorPins(SensorPins,SensorCount); // Set the sensor pins
sensor.SetTimeout(Timeout); // Set sensor timeout (us)
//Begin Serial Communication
Serial.begin(115200);
}
void loop()
{
sensor.ReadSensor(); // Read one full set of sensor values
// Cycle to through all current sensor values and print them
for(int i=0;i<sensor.GetSensorCount();i++)
Serial.printf("%d ",sensor.GetSensorValues(i));
Serial.println();
delay(30);
}
